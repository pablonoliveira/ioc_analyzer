⚙️ Instalação

1️⃣ Clonar Repositório

git clone https://github.com/seu-usuario/ioc_analyzer.git
cd ioc_analyzer

2️⃣ Criar Ambiente Virtual

python -m venv venv
venv\Scripts\activate

3️⃣ Instalar Dependências

pip install -r requirements.txt
💡 Nota: As APIs são opcionais.

🚀 Execução

python webapp.py

Acessar:

  • Local: http://localhost:5000
  • Rede: http://SEU_IP:5000

🌐 Acesso via Rede Local

Firewall Windows:

netsh advfirewall firewall add rule name="IOC Analyzer" dir=in action=allow protocol=TCP localport=5000
